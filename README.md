# 歌劇魅影：互動敘事

> *巴黎歌劇院，1881年。燭火搖曳，走廊深處傳來隱約的管風琴聲……*

以《歌劇魅影》為背景的互動小說。你扮演一位不起眼的芭蕾舞女伶，與克里斯汀·達厄並肩揭開魅影的秘密。每一個選擇都由你書寫，每一段對話都真實回應。

## 如何開始

### 第一步：取得 Anthropic API Key

本遊戲由 Claude AI 驅動，需要你自備 API Key。

1. 前往 [console.anthropic.com](https://console.anthropic.com/settings/keys)
2. 登入或註冊帳號（免費）
3. 點選 **Create Key**，複製產生的金鑰（格式為 `sk-ant-api03-...`）

> 新帳號通常附有免費額度，足以體驗完整劇情。

### 第二步：入場

開啟遊戲頁面後，會看到 API Key 輸入框。貼上你的金鑰，點擊「入場」。

- Key 只存在你的瀏覽器記憶體（sessionStorage），**關閉分頁後自動清除**
- 不會傳送給任何第三方，僅用於直接呼叫 Anthropic API

### 第三步：開始你的故事

輸入框會出現開場場景。直接用**自然語言**輸入你的行動或對話，例如：

```
「天使？你是說……他真的存在？」我盯著克里斯汀，試圖從她眼中看出玩笑的成分。
```

```
我悄悄往走廊深處望去，想看看剛才那個聲音是從哪裡來的。
```

```
我拉住她的手：「妳有沒有想過，那也許不是天使？」
```

## 遊戲說明

| | |
|---|---|
| **語言** | 繁體中文 |
| **視角** | 第二人稱（你） |
| **角色** | 歌劇院芭蕾舞女伶，克里斯汀的友人 |
| **風格** | 古典、戲劇、懸疑 |
| **操作** | Enter 送出，Shift+Enter 換行 |

**你可以做的事：**
- 與其他角色對話（克里斯汀、梅格、魅影……）
- 探索歌劇院的各個角落
- 做出影響劇情走向的選擇
- 為自己的角色取名、設定個性
- 偏離原著，走出屬於你的結局

點擊右上角「**離場**」可清除 Key 並重置故事。
